# Project Overview
Tiktok Downloader is a CLI build with node.js which aims to download videos of tiktokers.

### Setup

`npm install`

`npm link`

### Example

example of command line:

`tiktok-downloader --url https://www.tiktok.com/@aileenchristineee/video/6823846072973249797`
